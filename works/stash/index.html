How to install

Stash is written in Python, so the best way to go is to open up the terminal and type

pip install stash

That, of course, requires having pip installed.

We’ll write more stuff here lately as soon as the installer will be ready. There’ll be a separate set of instructions for each platform.

Type stash -h to check that stash is installed successfully. If this command produces sensible usage instructions, then we're all good.

If something goes wrong, please contact us—we'll be happy to help. Drop us a line at support@stashapp.com or ping our Twitter: @stashapp.

How to use

After installing, set up an account either on the web or in the app.

Log in

Type stash login to begin.

$ stash login
Please enter your email:
$ nyddle@gmail.com
Please enter the password:
$ *************
Logged in.

Create key

Stash keeps data as key-value pairs. Come up with a nice key name and put the meaningful stuff inside it.

$ stash 37signals_blog http://37signals.com/svn
Saved
$ stash 2read ‘About Face 3: The Essentials of Interaction Design’
Saved

General syntax is stash key_name value.

View

General syntax is stash get key_name or just stash key_name. 

$ stash get 37signals
http://37signals.com
$ stash 37signals
http://37signals.com

Edit

General syntax is stash set key_name new_value.

$ stash  set 37signals http://38signals.com
Saved

Delete

General syntax is stash (remove|delete|rm) key_name.

$ stash delete 37signals
Deleted

List keys

General syntax isstash show keysor juststash keys.

$ stash keys
nice_blogs
reading
37signals

Show all data

There are several commands to do this:stash (show|view|list|all).

$ stash all
nice_blogs: “Joel on Software, 37signals SVN”
reading: “About Face 3: The Essentials of Interaction Design”
37signals: “http://37signals.com”
