Diabase, the elegant and powerful cloud-based dialer solution. Diabase is a enterprise quality call center solution design for small teams made using the Twilio's voice platform.

Features

Campaign Management

Full campaign management and analysis allows you to create multiple campaigns, assign users and configure inbound queues to multiple campaigns.

Power dialing mode

Automatically dial through data records one at a time, dialbase will also automatically disposition any calls that are not a suitable use of your agents time (failed connections etc).

Call recording and monitoring

Record all calls (can be turned on and off to meet PCI/DSS needs). This feature allows you to make sure that all calls are conducted professionally.

Data Analysis

Organise your data sets to achieve the best possible results - settings and management sections help you effectively manage your data batches, giving you maximum data penetration.

Answer Machine Detection

ialbase detects answer machines before the call is put through to an agent and will disposition the call as an answer machine. Rules can be set to call these records back later.

Sales stats & campaign records

View live stats for individual campaigns or all campaigns stats for a clear picture of what is going on. Additional you can access historical call campaign data to monitor past performance.

Call disposition

Dialbase provides the ability for representatives to update each data record after they are finished with the call. These stats can be accessed at any time and stats are provided so you can see how data is performing.
